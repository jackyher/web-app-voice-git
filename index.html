Control de texto

Documentación

Este entorno de Interacción Humano-Computadora (IHC) permite identificar órdenes por medio de la voz utilizando el Web browser.

Instrucciones:

  • Para cambiar el tamaño del texto "Control de texto", di "Cambia a tamaño 3".
  • Para abrir una nueva pestaña en el navegador, di "Quiero abrir una nueva pestaña".
  • Para ir a la página de inicio de canva.com, di "Abre canva.com".
  • Para abrir una nueva ventana con un tamaño de 800 * 600, di "Cambia el tamaño de la ventana".
  • Para cerrar una pestaña del navegador, di "Quiero cerrar la pestaña actual".
  • Para aumentar el zoom a 150%, di "Quiero aumentar zoom".
  • Para reestablecer el zoom a 100%, di "Quiero reducir zoom".
